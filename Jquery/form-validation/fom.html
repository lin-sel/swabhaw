Form Validation

USERNAME:

  • username should contain at least one Alphabet letter
  • username should contain at least one Small letter
  • username should not contain white space
  • username should contain at least one Number
  • username should not be contain Any spcial character

PASSWORD:

  • password should contain at least on Alphabet letter
  • password should contain at least on Small letter
  • password should not contain white space
  • password should contain at least one Number
  • password should contain at least on Special Character % # $ @ & ! _ -
